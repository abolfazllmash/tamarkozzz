# بازی تمرکز (Stick Chaos / Avoid the Blocks)

یک بازی آرکیدِ مینیمالِ «جاخالی دادن» برای اندروید، نوشته‌شده با **Kotlin + Jetpack Compose**.
با کشیدن انگشت، دایره‌ی بازیکن را هدایت می‌کنی و باید در برابر بلوک‌ها، لیزرها، تیغه‌ها و دیگر
خطرها برای مدت هدفِ هر مرحله زنده بمانی.

---

## ساخت APK روی GitHub (خودکار)

این مخزن یک workflow آماده در `.github/workflows/android.yml` دارد:

1. کل پروژه را روی یک ریپوی گیت‌هاب push کن (شاخه‌ی `main`).
2. به تب **Actions** برو؛ workflow با نام **Build APK** خودش اجرا می‌شود
   (یا دستی با دکمه‌ی *Run workflow*).
3. بعد از اتمام، از بخش **Artifacts** همان اجرا، فایل `app-debug-apk` را دانلود کن.
   داخلش `app-debug.apk` است که مستقیم روی گوشی نصب می‌شود.

### انتشار نسخه‌ی ریلیز (اختیاری)

با push کردن یک تگ نسخه، علاوه بر دیباگ، APK ریلیز هم ساخته و در **Releases** منتشر می‌شود:

```bash
git tag v1.0.0
git push origin v1.0.0
```

اگر کلید امضای خودت را داری، این سکرت‌ها را در
`Settings → Secrets and variables → Actions` اضافه کن تا ریلیز با کلید واقعی امضا شود
(در غیر این صورت با امضای دیباگ ساخته می‌شود):

- `KEYSTORE_BASE64` — خروجی `base64 my-upload-key.jks`
- `STORE_PASSWORD`
- `KEY_ALIAS`
- `KEY_PASSWORD`

## اجرا روی سیستم خودت (Android Studio)

1. Android Studio را باز کن و **Open** را بزن و این پوشه را انتخاب کن.
2. بگذار Android Studio هنگام ایمپورت، Gradle و SDK لازم را همگام‌سازی کند.
3. روی شبیه‌ساز یا گوشی واقعی Run بزن.

> امضای دیباگ به‌صورت خودکار با کی‌استور پیش‌فرض اندروید انجام می‌شود؛
> دیگر نیازی به ساختن دستی `debug.keystore` نیست.
